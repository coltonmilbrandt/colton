My Portfolio

Gwin a Unique AMM

 Prototyped

This unique AMM that I coded allows users to create stable positions that provide liquidity, as well as long and short positions, without the need for a specific counterparty, order book, or contract length. The core functionality of the AMM can be used to create emulated assets from ETH, such as USD, Bitcoin, Gold, and Yen. This allows users to take advantage of the benefits of the Ethereum blockchain while still accessing the value and stability of these more traditional assets.
